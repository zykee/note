如何使用gitbook来做笔记

  1. 首先安装gitbook-cli
npm install gitbook-cli -g

  1. 然后创建自己专属的笔记文件夹

  2. 进入文件夹运行gitbook init 就会生成两个文件

  3. Readme.md 显示书的封面

  4. Summary.md是书的目录

  5. 启动服务器，查看和编辑书

gitbook serve

就可以在本地的4000端口访问 然后修改Sunmary.md就可以添加书的目录
